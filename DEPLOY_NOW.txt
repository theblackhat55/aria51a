⚡ QUICK START - Deploy in 3 Minutes:

1️⃣  AUTHENTICATE (30 seconds)
   
   Get your Cloudflare API token at:
   👉 https://dash.cloudflare.com/profile/api-tokens
   
   Click "Create Token" → Use "Edit Cloudflare Workers" template
   Required permissions: Pages:Write, D1:Write, Workers:Write
   
   Then run:
   export CLOUDFLARE_API_TOKEN="your-token-here"

2️⃣  DEPLOY (2-3 minutes)
   
   cd /home/user/webapp
   ./deploy-production.sh

3️⃣  VERIFY (1 minute)
   
   Open: https://aria51.pages.dev/health
   Login: https://aria51.pages.dev/login (admin / demo123)
   Test: https://aria51.pages.dev/risk (should show 8 risks)
